# AI-Powered Personal Finance Dashboard: Learning Resources

This document is a curated collection of resources to help you master the technologies used in this project. Use these to deepen your understanding as you move through the roadmap.

## 🏛️ Phase 1: The Core Stack (Django & React)

### Django (Backend)
- **[Official Django Tutorial](https://docs.djangoproject.com/en/stable/intro/tutorial01/):** The best place to start for understanding Models, Views, and the ORM.
- **[Django REST Framework Guide](https://www.django-rest-framework.org/):** Learn how to build the JSON APIs that your React frontend will consume.
- **[Simple is Better Than Complex](https://simpleisbetterthancomplex.com/):** High-quality tutorials for specific Django implementation challenges.

### React (Frontend)
- **[React Documentation (Beta)](https://react.dev/):** The modern, interactive way to learn React hooks and components.
- **[The Joy of React](https://www.joyofreact.com/):** A highly recommended visual guide to mastering React state and UI patterns.
- **[Vite Guide](https://vitejs.dev/guide/):** The build tool we'll use to initialize the frontend.

---

## 🏗️ Phase 2 & 3: System Design & Microservices

### Architecture & Patterns
- **[Microservices.io](https://microservices.io/):** Comprehensive patterns for distributed systems (API Gateway, Event Sourcing, etc.).
- **[ByteByteGo (YouTube)](https://www.youtube.com/@ByteByteGo):** Visual breakdowns of high-scale system design.
- **[Full Stack Python](https://www.fullstackpython.com/):** Explains the entire Python web ecosystem from servers to task queues.

### Background Tasks & AI
- **[Celery First Steps](https://docs.celeryq.dev/en/stable/getting-started/first-steps-with-celery.html):** Learn how to handle long-running Python logic (like AI) asynchronously.
- **[FastAPI Tutorial](https://fastapi.tiangolo.com/tutorial/):** A guide to the high-performance framework used for our AI microservice.
- **[Hugging Face NLP Course](https://huggingface.co/learn/nlp-course/):** Learn how to process text using modern Python AI models.

---

## 🐳 Phase 4: DevOps & Deployment

### Containerization
- **[Docker Curriculum](https://docker-curriculum.com/):** Hands-on tutorial for Docker and Docker Compose.
- **[TestDriven.io - Dockerizing Django/React](https://testdriven.io/blog/dockerizing-django-with-postgres-gunicorn-and-nginx/):** A specific guide that mirrors our project's architecture.

### Engineering Standards
- **[Refactoring.Guru](https://refactoring.guru/):** Learn clean code and design patterns to keep your codebase maintainable.
- **[Pytest Documentation](https://docs.pytest.org/):** The industry standard for testing Python applications.
